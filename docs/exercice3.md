# Exercice 3 - Reflexion ethique

## Risque identifie
Un patient pourrait se soigner seul sans consulter un medecin.

## Propositions d'amelioration

1. **Avertissement visible** : Ajouter un bandeau rouge dans l'interface : '⚠️ Ceci n'est pas un diagnostic medical. Consultez toujours un medecin.'

2. **Renforcement du prompt** : Modifier SYSTEM_PROMPT pour ajouter : 'Tu dois toujours conclure par un rappel de consulter un medecin.'
